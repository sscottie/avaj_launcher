## AVAJ_LAUNCHER

Симуляция аэропорта.

Прилагается сценарии запуска летающих средств в двух вариантах:
1. Стандартный;
2. С использованием md5 хешированных названий Летательных средств;

Для запуска:
```console
$ bash ./launch.sh
```
или
```console
$ bash ./launch_md5.sh
```

Посмотреть результаты симуляции:

```console
$ cat simulation.txt
```

Очистить от .class файлов:
```console
$ bash ./clear_class.sh
```
